# 发布 1.2.1 设计

## 目标

为缩短宽屏初始化保护的修复发布补丁版本 `1.2.1`。

## 变更范围

- 将 `bilistudy.user.js` 的 userscript 元数据 `@version` 从 `1.2.0` 更新为 `1.2.1`。
- 将 README 的当前版本和 jsDelivr 固定版本链接从 `1.2.0` / `v1.2.0` 更新为 `1.2.1` / `v1.2.1`。
- 提交版本变更后，在该提交上创建带注释标签 `v1.2.1`。
- 将 `main` 和 `v1.2.1` 推送到 `origin`。

## 验证

- JavaScript 语法检查通过。
- 仓库内不再存在作为当前发布信息的 `1.2.0` 或 `v1.2.0`。
- 工作区干净，本地 `main` 与 `origin/main` 一致。
- 本地及远端均存在指向版本提交的 `v1.2.1` 标签。
